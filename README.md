# Parking Management
This project is front-end of parking management system.

## How to Run:
#### Requirement:
- Installed: Nodejs (v20.8.0), yarn
#### Steps
Step 01: Clone project. <br />
Step 02: Run the following commands:
```
yarn install
yarn dev
```
